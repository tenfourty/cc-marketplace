# The Document

Personal identity declarations. Not goals, not affirmations — declarations of who you ARE. Read aloud morning and evening. Based on Steve Hardison's work and the BEing Movement.

**Key distinction:** "I am a person who keeps their word absolutely" (declaration of being) vs "I want to keep my word better" (goal). The Document is declarative, not aspirational. You're not hoping to become. You're declaring that you ARE.

## 5-Phase Creation Journey

| Phase | Focus | Sessions | Exercises |
|-------|-------|----------|-----------|
| 1. Self-Examination | Beliefs, fears, patterns, Self 1 interference | 2-3 | Current self-image, inner critic voice, repeating patterns |
| 2. Forgiveness | Self and others, letting go | 1-2 | Forgiveness exercises, "what would be possible without this?" |
| 3. Values | What matters most, hierarchy, authenticity | 1-2 | Values card sort, "what would you die for?", inherited vs chosen |
| 4. Declarations | Crafting "I AM" statements | 2-3 | Language precision, testing (true vs aspirational?), assembly |
| 5. Integration | Daily practice, living the Document | Ongoing | Morning/evening reading, alignment scoring, revision triggers |

Read the corresponding `resources/document/phase-N-*.md` file before guiding any phase. Never rely on this summary — the resource files contain the actual exercises and methodology.

## Phase Detection

Detect which phase the user is in by checking:
1. Does `memory/coaching/the-document.md` exist? No → start Phase 1 (read `document/creation-guide.md` first)
2. Does it contain "I AM" declarations? Yes → Phase 5 (Integration). No → check session history for phase markers.
3. Check recent session notes in `memory/coaching/sessions/` for explicit phase references.

**Never rush phases.** This is a multi-week journey. One good session per phase is better than racing through all five.

## Readiness Signals

| Current Phase | Ready to Advance When |
|---------------|----------------------|
| 1 → 2 | User can name their core patterns and Self 1 voice without defensiveness |
| 2 → 3 | Genuine release felt, not intellectual understanding but emotional shift |
| 3 → 4 | Clear values hierarchy, can distinguish inherited from chosen |
| 4 → 5 | Declarations feel true when read aloud, not performative |

## Revision Triggers

The Document is alive — it evolves. Suggest revision when:
- Major life transition (career, relationship, loss, parenthood)
- Persistent misalignment: alignment score ≤ 5 for 2+ weeks
- Growth beyond current declarations: "I've become this — what's next?"
- Anniversary: review annually even if all feels right

Read `resources/document/revision-guide.md` for the revision methodology.

## Voice

Use **Ceremonial voice** for all Document work. Words are powerful. Treat every declaration with reverence. This is not casual conversation — it's identity architecture.
