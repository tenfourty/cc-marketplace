## Overview

Values are the third phase of creating The Document. Before declarations can be crafted, you need to know what you stand for — not what sounds good, not what you were told to value, but what you would actually sacrifice for. Values are the foundation from which declarations are built. A declaration that isn't grounded in a genuine value will feel performative when read aloud. Identifying values is not about aspiration — it is about honest excavation of what already matters most.

## When to Use

- User has completed Phases 1 and 2 and is ready for Phase 3
- User is drafting declarations that feel empty or generic — often a sign values work is incomplete
- User cannot distinguish between a declaration and a wish — values work grounds declarations in what's real

## When NOT to Use

- Phase 1 or Phase 2 is incomplete — values surfaced without prior self-examination may be inherited or performed rather than genuine
- User is highly activated or in a difficult emotional state — this phase requires reflective capacity

## Estimated Duration

1-2 sessions, with reflection time between. Allow 1-2 weeks. Some values reveal themselves over time, not all at once.

## The Core Distinction: Inherited vs Chosen Values

Most people carry a mixture of inherited and chosen values — and many have never examined which is which.

**Inherited values** are values absorbed from family, culture, religion, or early environment. They feel like "the way things are" rather than chosen positions. They can be genuine and deeply held. They can also be constraints that were never consciously accepted.

Examples of values that are often inherited:
- "Work hard and don't complain" (family culture)
- "Success means financial security" (class background)
- "Don't make a scene / keep the peace" (conflict avoidance modeled in childhood)

**Chosen values** are values you would stand behind if you examined them fully, in the light of your current adult understanding. Some inherited values survive this examination and become chosen. Others do not.

The purpose of this distinction is not to reject inherited values — it is to ensure that declarations are built on what you actually stand for, not on what you absorbed by default.

The question to ask: "If I were free to choose, would I choose this value?" If yes, it becomes chosen. If no, it belongs to someone else's Document.

## Core Exercises

### Exercise 1: The Values Longlist
Generate a complete, uncurated list of values — everything that matters to you, stated in whatever form comes naturally. Do not filter or rank yet. Include things that feel obvious and things that feel uncomfortable to claim.

Prompts to generate the list:
- "What do you believe should be honored in how people treat each other?"
- "What do you consistently prioritize, even when it costs you?"
- "What violations — in yourself or others — produce the strongest reaction in you?"
- "Complete this: Life is too short to [______] or not to [______]."
- "What do you most admire in other people? What does that reveal about what you value?"
- "What would you tell your children (or the young people you care most about) to stand for?"
- "What have you found yourself defending most passionately in the last year?"

A typical longlist is 20-40 items. Common values include: integrity, courage, loyalty, freedom, creativity, excellence, service, family, honesty, justice, growth, compassion, independence, beauty, discipline, presence, humor, adventure, community. But the list must come from the user — do not suggest a standard list.

### Exercise 2: Narrowing to the Core
From the longlist, guide the user through a progressive narrowing.

**Round 1 — Top 10:** Ask them to identify the ten values that feel most essential. When choosing, use this test: "If this value were absent from my life, something central would be missing." Not "this would be unfortunate" but "this would be a fundamental loss."

**Round 2 — Top 5:** From the ten, narrow to five. This is harder. The test becomes more demanding: "This is something I would sacrifice other things for. I have already sacrificed for this, even when it cost me."

**Round 3 — Rank the Five:** Order them 1-5. This requires distinguishing what you value most from what you also value. There is no right answer. The ranking is a tool for understanding your own hierarchy, which determines what wins when values conflict.

### Exercise 3: The Sacrifice Test
For each value in the top five, apply the sacrifice test. This distinguishes genuine values from preferred values.

Ask:
- "Can you name a time when you honored this value when it cost you something?"
- "Is there a time you compromised this value and it still troubles you? What does that tell you?"
- "If honoring this value required giving up something else you care about, would you?"

If the user cannot name a specific instance of sacrifice or cost, the value may be aspirational rather than actual. That is worth noting — it may mean this is a value they are moving toward, not one they currently stand in.

### Exercise 4: When Do You Feel Most Alive?
Values are not just ethical positions — they are the conditions under which you are most fully yourself. This exercise surfaces values through the felt experience of aliveness.

Ask:
- "Describe a time in your life when you felt most alive, most yourself, most in your element. What was happening?"
- "What was present in that experience that isn't always present?"
- "What would need to be true about your daily life for that feeling to be more frequent?"

The qualities present in those moments of aliveness are often values in action. Name them explicitly: "It sounds like freedom was present there. And creative expression. Are those on your values list?"

### Exercise 5: Distinguishing Inherited from Chosen
For each value in the top five, run this examination:

- "Where did this value come from? Can you trace it?"
- "If you had grown up in a completely different family and culture, do you think you would still hold this value?"
- "Have you ever questioned this value — and after questioning it, kept it? Or have you simply never questioned it?"
- "Is there any part of holding this value that is about loyalty to a person or a place rather than genuine conviction?"

Values that survive this examination with a clear "yes, I choose this" are the ones to build declarations on.

## The Values Hierarchy in Practice

The hierarchy matters when values conflict — which they inevitably will. A person who ranks freedom first and family second will make different decisions under pressure than one who ranks them in reverse order. Both are valid. Neither is better. But they must be known.

Ask: "Imagine a situation where your top two values directly conflict with each other. What happens? Which one wins?" This usually confirms or revises the ranking.

## Key Questions

- "What do you stand for — not what you want to stand for, but what you actually stand for right now?"
- "What values have you inherited that you are keeping, and which are you ready to hand back?"
- "Which of your values have been tested by real circumstances? Which are untested?"
- "Where in your life are you currently violating your own values? What does that cost you?"
- "What would your declarations sound like if they were built entirely on these five values?"

## Completion Criteria

Phase 3 is complete when:
1. The user has a ranked list of their top 5 values, arrived at through genuine reflection — not selected from a pre-printed card deck
2. Each value has been tested against the sacrifice test — the user can name specific instances where they lived this value under cost
3. The user has explicitly examined which values are inherited and made a conscious choice about each
4. The user can articulate why each value is in the hierarchy in the order it is — the ranking is understood, not arbitrary
5. There is a clear sense of which values will anchor the declarations in Phase 4

## Pitfalls

**Selecting aspirational values.** The user chooses the values they want to have rather than the ones they currently live. The sacrifice test catches this — if there is no history of cost, the value is aspirational. Aspirational values belong in declarations only if the user consciously chooses to declare them; they should not be presented as existing reality.

**Inherited values without examination.** Many people take their top values directly from their upbringing without ever questioning them. This is not always wrong — the values may be genuinely chosen upon examination. But they must be examined, not assumed.

**Too many top values.** When everything is a top value, nothing is. Push back if the top five contains ten things. Ask: "If you could only keep three, which three?" This forces genuine prioritization.

**Treating values as personality traits.** Values are not personality descriptions. "I value being organized" describes a preference. "I value integrity" describes a commitment. Distinguish between how you tend to be and what you stand for.

**Avoiding the ranking.** Some users resist ranking their values because it implies some matter more than others, which feels uncomfortable. Reframe: "This isn't about which values are better. It's about what wins when they compete." The ranking is practical, not moral.
